Volviendo a la definición

¿Qué es Docker?

Docker es un proyecto de código abierto que automatiza el despliegue de aplicaciones dentro de contenedores de software, proporcionando una capa adicional de abstracción y automatización de virtualización a nivel de sistema operativo en Linux.

Registry y Docker Hub

Según Docker...

The Registry is a stateless, highly scalable server side application that stores and lets you distribute Docker images.

The Registry is open-source, under the permissive Apache license.

  • Un Registry es un repositorio únicamente de imágenes de Docker.
  • Este repositorio nos permite actualizar las versiones de las imágenes, y a su vez distruirlas.
  • Podes levantar tu propia Registry con Docker.

Docker Hub

Docker Hub es la principal plataforma de almacenamiento de imágenes de Docker adoptada por la comunidad.
